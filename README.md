# Trois（トロワ）

> 近くの3軒だけ、AIが選ぶ。

店が多すぎて選べない。そんな「決められないストレス」をなくすための iPhone アプリ。
現在地・徒歩◯分・予算・気分を伝えるだけで、AIが付近のお店を **3軒だけ、理由つきで** 提案する。
候補を並べないから、迷わない。"あえて絞る" から、すぐ決まる。

## このアプリの正体

検索アプリではなく **選択（決定）アプリ**。
ユーザーがやりたいのは「店を探す」ことではなく「決めるのを誰かに肩代わりしてほしい」こと。
このアプリの価値は **決断疲れを減らすこと** であり、"3軒に絞る" はその価値の最適な表現。

名前 `Trois` はフランス語で「3」。
- 提案数の **3軒**
- 将来対応しうる **朝・昼・夜の3食**
の両方に掛かるダブルミーニング。機能を足すほど名前の意味が強化される設計になっている。
