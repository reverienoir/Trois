# 06. デザイン設計

## カラーパレット

| 役割 | カラー名 | コード |
|---|---|---|
| メイン | セージグリーン | `#7A9E7E` |
| アクセント | テラコッタ | `#D4845A` |
| 背景 | クリーム | `#F5F0E8` |

### 選定理由

- **赤・オレンジ系を避けた理由**: 食欲刺激系グルメアプリ（マクドナルド・ドミノピザ等）と同じ文脈になるため。Troisは食欲を刺激するアプリではなく、迷いを減らすアプリ。
- **青系を避けた理由**: 世の中に青系アプリが多すぎる。銀行・ビジネスツール感も強い。
- **セージグリーンを選んだ理由**: 「落ち着かせる」という目的に合っている。Calm・Headspaceなど、ユーザーの状態を整えるアプリに多用される色系統。「疲れてる」「健康的に」という気分選択とも世界観が一致する。
- **テラコッタ**: 食・温かみとの親和性を残しつつ落ち着いた印象。ゴールドより押しつけがましくない。
- **クリーム背景**: 白より柔らかく、セージ・テラコッタとの相性が良い。

### 参考にした世界観
- Calm / Headspace（落ち着かせる系アプリの色使い）
- 近年のAppleデザイン傾向（ナチュラル・セージ系）
